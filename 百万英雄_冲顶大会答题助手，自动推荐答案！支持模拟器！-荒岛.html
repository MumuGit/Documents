百万英雄/冲顶大会答题助手，自动推荐答案！支持模拟器！

2018-01-14 分类：学无止境 阅读(496) 评论(2)

目前网上不知道又是刮起了一波什么风，非常多的APP厂商都在搞这种“撒币游戏”，说白了就是答题送钱。玩法很简单，下载他们的APP，登录你的账号，准点进入直播间参与答题就行，不过要想获得奖励必须要全部答对。

虽然这类活动出的题目都不算难，并且就拿“百万英雄”来说一场题目也就12题，题数也并不多，但难免会有一些角度刁钻的问题出现，这时候想百度？不好意思每题作答时间就10s，估计你题目还没打出来就已经下一题了。。。

这时候就有一些程序员们站出来了，他们开发了一系列的辅助程序用来帮助用户作答。不过就目前市面上的这些辅助程序而言，这类程序真的还只能算是“辅助”。因为作答正确率普遍都不高，很多时候还是需要人脑来进行判断的。

下面LALA就在Github上找到了一个用Python写的百万英雄/冲顶大会答题助手程序，抱着玩玩看的态度来部署下这个程序，看看实际的作答效果如何。

首先是准备工作，这个答题助手和之前的微信跳一跳所需要的环境类似：

1、夜神模拟器：https://www.yeshen.com/

2、Python3：https://www.python.org/

3、将夜神模拟器的内置adb添加到系统环境变量

如何安装并配置这两款软件，请看我之前的这篇文章：

微信跳一跳辅助，全自动跳，解放双手！

在做完上述工作后，接下来我们就来注册百度OCR，百度OCR是个什么东西？简单点说就是智能AI，内置了一个文字识别的服务，而此款辅助工具就依靠此服务来工作。个人用户注册，每天都可以免费使用500次识别。

注册地址：http://ai.baidu.com/tech/ocr/general

按如图操作来创建一个应用：

填写“应用名称”和“应用描述”即可，其他地方都默认别做改动，然后点击立即创建：

创建好了后你可以得到三个有用的东西：AppID、API Key、Secret Key

妥善保管这三个参数，待会要用到。

现在我们来部署答题助手。

下载答题助手程序：https://github.com/smileboywtu/MillionHeroAssistant

如何在Github上下载？看下图：

解压压缩包到任意盘符，我这里就是H盘，此时打开CMD，用CMD进入到H盘并CD到程序的目录下：

H:
cd MillionHeroAssistant-master

如图：

安装程序所需依赖：

pip install -r requirements.txt

注：由于国内网络太辣鸡，这里如果下载速度过慢或者断开，建议开个SS或者VPN。

右键使用IDLE打开你程序目录中的config.py文件：

将如图中红框部分的参数修改成我们之前自己注册的：

退出IDLE并保存：

此时我们就可以打开夜神安卓模拟器了，等待模拟器启动成功，回到CMD中键入如下命令：

adb devices

如果回显出设备IP和端口就说明adb服务是正常的，这样就可以保证答题助手正常的运行：

接着我们在模拟器内下载要答题的APP，这里我用西瓜视频做演示好了，打开西瓜视频后，回到CMD中输入如下命令启动辅助程序：

python main.py

西瓜视频的活动名称是“百万英雄”，所以这里我们输入1并回车：

届时辅助程序会打开你电脑内的Chrome并访问百度，该Chrome界面上提示如图信息就说明辅助正常运行：

当“百万英雄”活动开始的时候，界面显示问题的时候，此时你在CMD内键入回车，辅助程序就会自动帮你在百度搜索近似答案：

比较遗憾的是，程序的作答成功率和之前我猜想的一样，不是特别准确，比如下面这个题目就答错了：

到这里整个辅助就部署完成了，你可以安静的等待下一场比赛开始，然后就可以使用此答题助手来辅助作答了。但你需要注意的是，辅助程序务必在每场答题比赛开始前就启动好。

写在最后：

在部署完成后，我大致也了解这个辅助的工作原理了：

当用户按下回车后，首先程序会利用adb调试功能，截取当前模拟器的屏幕，然后利用百度OCR文字识别将图片中的文字自动输入到百度中进行搜索，接着通过一个“公式”来计算搜索结果中相关的内容，将出现频率最高的作为近似答案回显在CMD中。

所以当你知道工作原理是这样后，也就能够理解为什么这些辅助在很多时候给的答案都不准确了，毕竟很多冷门、刁钻的问题在百度通过问答的关键字来进行搜索本身搜出来的结果就不可能正确，其次能不能搜到相关问题都还是一个未知数，所以这款辅助程序，它真的只能算是一款“辅助工具”，还上升不到外挂的级别。

最后，这些APP举办的活动大家别太当真，开个心，娱乐下就好，没有商家是SB整天做赔本的买卖，用几百万来吸引上千万的装机量，这本身就是一个稳赚不赔的套路了，其次放长线钓大鱼这个道理，我想大家都懂~
