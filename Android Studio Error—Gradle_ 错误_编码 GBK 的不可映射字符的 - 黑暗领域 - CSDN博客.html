Android Studio Error—Gradle: 错误:编码 GBK 的不可映射字符的

标签： android studioGradle出错GBK中文
2013-09-24 15:09 28714人阅读 评论(27) 收藏 举报
分类：

版权声明：本文为博主原创文章，转载请附带本文链接，这是大家原创的动力，谢谢

代码中的中文注释可能会引起Gradle报错


解决办法：
在项目下的build.gradle下添加以下代码即可解决
 

[java] view plain copy
 print?
  1. tasks.withType(Compile) {  
  2.     options.encoding = "UTF-8"  
  3. }  
tasks.withType(Compile) {
    options.encoding = "UTF-8"
}

Gradle2.0+环境下需将Compile改为JavaCompile

[java] view plain copy
 print?
  1. tasks.withType(JavaCompile) {  
  2.     options.encoding = "UTF-8"  
  3. }  
tasks.withType(JavaCompile) {
    options.encoding = "UTF-8"
}
